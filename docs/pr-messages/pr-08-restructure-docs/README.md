# Restructure documentation into theming and components subfolders

This pull request restructures and improves the documentation for the `@littlebranches/giselle-mui` library. The changes focus on reorganizing content into a clearer hierarchy, adding frontmatter for sidebar navigation, and improving discoverability and usability of the docs.

**Documentation structure and navigation improvements:**

- Added frontmatter (sidebar position and label) to all documentation files to enable sidebar navigation and improve organization.
- Moved and renamed documentation files to group them by topic (e.g., theming, components), creating a more logical and user-friendly structure.

**Content updates:**

- Updated the main `README.md` to provide an overview, installation instructions, and links to theming and component documentation.
- Added or updated component-specific and theming documentation to reflect the new structure and provide clearer guidance.

These changes make the documentation easier to navigate and maintain, and help users quickly find relevant information.
