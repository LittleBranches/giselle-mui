## Summary

Adds `docs/pr-messages/README.md` ‚Äî a master PR index covering all 25 PRs in this repository from inception through to the open career-timeline branch (PR 25).

Also adds stub redirect files for every PR that previously had no companion document, a `docs/README.md` index entry, and a `.gitignore` entry for the Vitest `coverage/` output directory.

---

## What Changed

### New: `docs/pr-messages/README.md`

Master index with four sections:

1. **Branch To Roadmap Mapping table** ‚Äî one row per source branch; columns: branch name, PRs, commits + tier, dates (opened ‚Äì merged), roadmap phase(s), milestone summary.
2. **Roadmap Milestone To Branch Mapping** ‚Äî vice-versa lookup; one row per roadmap phase, listing all contributing branches.
3. **Branches** ‚Äî `###` section per branch with a two-sentence context description and bulleted PR list with GitHub links and companion doc links.
4. **Tag and commits-tier legends** ‚Äî `[ENABLER]`, `[QUALITY/REVIEW]` tags explained; XS/S/M/L/XL commit tier thresholds defined.

Covers all 25 PRs across 17 source branches, including the open PR 25 (`feature/giselle-mui-career-timeline-finalisation`) which is listed separately from `feature/giselle-theme-preset`.

### New: per-PR stub redirect files

For every PR that had no companion document, a one-line stub was added pointing to the nearest existing context document. Affected PRs: 1, 2, 3, 4, 5, 6, 7, 8, 10, 11, 13, 15, 18, 19, 20, 21, 22, 23.

### New: multi-PR folder `README.md` files

`pr-12/README.md`, `pr-14/README.md`, `pr-16/README.md`, `pr-17/README.md`, `pr-24/README.md`, `pr-25/README.md` ‚Äî index files for branches that produced multiple companion documents.

### Updated: `docs/README.md`

Added two entries to the docs index:

- `[Documentation strategy](./documentation-strategy.md)`
- `[PR messages index](./pr-messages/README.md)`

### Updated: `.gitignore`

Added `coverage/` to prevent Vitest's lcov HTML report from appearing as untracked files.

### Updated: `docs/incidents/timeline-hover-regression-may-2026.md`

Minor typo fix (one character, pre-existing).
